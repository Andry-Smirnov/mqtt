Reference for unit 'mqttpackets': Procedures and functions

ReadBinaryDataFromBuffer

  

Reads binary data from a buffer in MQTT protocol format. Used to store password

ReadMQTTPacketFromBuffer

  

Reads the packet type, flags and remaining length of a packet from a buffer. Performs validation and returns an error code. Creates the appropriate packet class and calls that packet class to finish reading the packet from the buffer.

ReadRemainingLengthFromBuffer

  

Reads the remaining length of an MQTT packet from a buffer, which is stored in a special multi-byte format

ReadUTF8StringFromBuffer

  

Reads a UTF8 String from a buffer in MQTT protocol format

ReadWordFromBuffer

  

Reads a word from buffer in network byte order
