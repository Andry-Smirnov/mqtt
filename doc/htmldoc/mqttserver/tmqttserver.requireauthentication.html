TMQTTServer.RequireAuthentication

Require clients to provide a valid username and password to connect

Declaration

Source position: mqttserver.pas line 195

published property TMQTTServer.RequireAuthentication : Boolean
  read FRequireAuthentication
  write FRequireAuthentication
  default True;

Description

If false, a username and password will be validated if it is provided but connections where no username and password are provided will also be allowed. If a username and password are provided, the session will be associated with a user object. The default is false.
